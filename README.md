changelog-md-cli
================
 A CLI for the `changelog-md` module (better description forthcoming...)

Install
-------
```sh
$ npm install --save changelog-md-cli
```

Usage
-----
```sh
$ changelog-md # Writes a base changelog to stdout.
```

Testing
-------
```sh
$ npm test
```
